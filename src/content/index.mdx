# Kiro 워크샵

### 개요
Kiro는 AI 기반 통합 개발 환경으로, 강력한 AI 지원 개발 기능을 제공하여 코드 작성, 디버깅, 유지보수를 더 효율적으로 할 수 있도록 도와줍니다. 
이번 워크샵은 Kiro IDE의 핵심 기능들을 1.5시간 안에 익힐 수 있도록 설계된 집중형 학습 과정입니다. Kiro를 효과적으로 사용하기 위한 필수 기능들을 엄선하여 실습 중심으로 구성했습니다.

### 학습 목표
2시간 후 다음을 할 수 있게 됩니다:

- Kiro IDE 기본 사용법 이해
- 효과적인 AI 프롬프팅 기법 습득
- Steering을 통한 개발 표준 설정
- Vibe 세션 vs Spec 세션 차이점 체험 ⭐
- Agent Hooks를 통한 개발 워크플로우 자동화
- 상황별 적절한 개발 방법론 선택 ⭐
- MCP를 통한 외부 도구 연동

### 학습 모듈
##### 바이브 세션 실습 (30분)
- 모듈 1: 환경 설정 및 첫 실행 (20분)
- 모듈 2: 효과적인 프롬프팅 (10분)
- 모듈 3: Steering 설정 (10분)

##### 스펙 세션 실습 (80분)
- 모듈 4: Vibe vs Spec 세션 비교 ⭐ (30분)
- 모듈 5: MCP 기본 사용법 (10분)
- 모듈 6: Agent Hooks 자동화 (10분)

##### 추가 실습 (선택사항)
- 모듈 7: 다음 단계 및 추가 학습
- 모듈 8: 워크샵 요약

##### 사전 준비
- 컴퓨터 (Windows, macOS, Linux)
- 웹 브라우저
- 인터넷 연결
- AWS 계정 (루트 사용자 또는 관리자 권한)
- 유효한 이메일 주소 (@gmail.com 또는 @naver.com)
- Google Authenticator(iOS) 또는 Google OTP(Android) 앱 설치

### 워크샵 구조
