- AI 네이티브: 처음부터 AI와의 협업을 염두에 두고 설계
- 컨텍스트 유지: 프로젝트 정보를 지속적으로 기억
- 구조화된 개발: Spec Driven Development 지원
- 자동화 지원: Agent Hooks를 통한 워크플로우 자동화
- 확장 가능: MCP를 통한 외부 도구 연동

##### 🚀 Vibe 세션 (대화형 개발)

- 빠른 질문과 설명
- 대화형 프로젝트 구축
- 탐색적 코딩

##### 📋 Spec 세션 (구조화된 개발)

- 요구사항 → 설계 → 구현
- 체계적인 개발 프로세스
- 철저한 구현 보장

##### ⚙️ Steering (개발 표준)

- 팀 코딩 스타일 자동 적용
- 일관된 코드 품질 유지

##### 🔧 Agent Hooks (자동화)

- 반복 작업 자동화
- 코드 품질 자동 검사

##### 🔗 MCP (외부 연동)

- 실시간 웹 정보 접근
- 외부 도구 통합
