### ⚒️ 워크샵 스튜디오 로그인 단계

1. [스프레드 시트](https://docs.google.com/spreadsheets/d/1IBOqaO8CxNKGkwRm2JxkqelJs71T2C6h6lsB46G_jlM/edit?usp=sharing)
에서 본인의 이름에 해당하는 계정 링크를 클릭하여 Workshop Studio 로그인 화면으로 이동합니다.

![1](/img/workshop_event_login/1.png)

2. 로그인 화면에서 Email one-time password (OTP)를 선택합니다.

<img className="w400" src="/img/workshop_event_login/2.png" />

3. 본인의 이메일 주소 (⚠️ **@gmail.com** 또는 **@naver.com**)를 입력하고 Send passcode를 선택합니다.

![3](/img/workshop_event_login/3.png)

4. 본인 이메일 주소로 패스워드(9자리 숫자)가 도착하면 Passcode에 입력하고 Sign in을 선택합니다.

![4](/img/workshop_event_login/4.png)

5. "agree with the Terms and Conditions"를 체크하고 Join Event를 선택합니다.

<img className="w500" src="/img/workshop_event_login/5.png" />

6. Open AWS console을 선택하여 AWS Console에 접속합니다. (중국어 및 영어 설명은 무시합니다.)

![6](/img/workshop_event_login/6.png)

7. 다음과 같이 AWS Console 화면에 정상적으로 진입했다면 모든 준비가 끝났습니다.

![7](/img/workshop_event_login/7.png)

> [!NOTE]
>
> 🚨 모든 실습은 us-east-1 (버지니아 북부) 리전에서 진행합니다. us-east-1 (버지니아 북부)가 아닌 다른 리전으로 변경할 경우 실습이 불가합니다.
